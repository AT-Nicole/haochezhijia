# 豪车之家 技术架构文档

## 1. 项目概述

豪车之家是一个大湾区高端汽车服务平台，复刻豪车毒商业模式，对标并超越汽车之家。
平台采用撮合模式，自身不持有车辆，连接买家与车行，通过AI全自动化运营降低人力成本。

## 2. 技术选型

| 层级 | 技术方案 | 理由 |
|------|---------|------|
| 前端框架 | Taro 4.0 + Vue 3 | 微信小程序 + H5 跨端，组件化开发 |
| 后端服务 | 微信云开发 | 免服务器/免域名/免运维，SDK直连 |
| 数据库 | 云开发数据库（MongoDB） | 免运维，自动扩缩容 |
| 云存储 | 云开发存储 | 图片/视频/文件上传 |
| AI能力 | 大模型API（DeepSeek） | 成本低（¥1/百万token），中文能力强 |
| 认证 | 微信登录 | 免注册，一键授权 |
| 支付 | 微信支付 | 小程序内完成交易闭环 |

```
    │
    └──▶ 管理员 → 后台/展厅/会员/财务/我的
```

## 9. 部署流程

1. 注册微信小程序（获取 AppID）
2. 开通微信云开发（创建环境）
3. 修改 `project.config.json` 中的 `appid`
4. 修改 `src/services/cloud.js` 中的 `ENV_ID`
5. 在云开发控制台创建数据库集合（按 database-design.md）
6. 部署云函数（login / vehicleService / aiService 等）
7. 设置定时触发器
8. `npm install && npm run dev:weapp` 本地预览
9. 微信开发者工具上传 → 提交审核 → 发布
